# PocketSplit

**PocketSplit** is a mobile-first web application designed to simplify daily expense tracking and bill splitting. Built for speed and ease of use, it allows users to manage personal finances and share costs directly from their browser without the need for accounts or backend servers.

## Features
-   **Expense Tracking**: Log daily expenses with amount, category, and notes.
-   **Smart Dashboard**: View real-time summaries of daily and monthly spending.
-   **Bill Splitter**: Instantly calculate per-person amounts, including tips, for shared bills.
-   **Local Persistence**: Data is stored securely in the browser's LocalStorage, ensuring privacy and continuity.
-   **Mobile-First Design**: A responsive, app-like experience optimized for mobile devices.

## Tech Stack
-   **Frontend**: React, Vite
-   **Language**: JavaScript (ES6+)
-   **Styling**: Tailwind CSS
-   **State Management**: React Context & Hooks
-   **Deployment**: Vercel
